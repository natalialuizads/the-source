# Git Flow

## Comandos Essenciais

```bash
# Iniciar repositório
git init

# Clonar
git clone <url>

# Status
git status

# Adicionar mudanças
git add .

# Commit
git commit -m "mensagem"

# Enviar
git push origin <branch>

# Puxar atualizações
git pull origin <branch>
```

## Branching
- `main`: Produção.
- `develop`: Desenvolvimento.
- `feature/nome`: Novas features.
- `hotfix/nome`: Correções urgentes.
