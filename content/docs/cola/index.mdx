# Cola (Cheatsheet)

Acesso rápido a configurações e snippets recorrentes.

## Git

```bash
git commit -m "feat: nova funcionalidade"
git push origin main
git checkout -b feature/nova-feature
```

## Linux Terminal
- `ls -la`: Listar todos os arquivos com detalhes.
- `grep -r "texto" .`: Buscar texto recursivamente.
- `chmod +x script.sh`: Tornar executável.

## VS Code Shortcuts
- `Ctrl + P`: Buscar arquivo.
- `Ctrl + Shift + F`: Buscar no projeto.
- `Ctrl + '`: Abrir terminal.
