# Next.js

Framework React para produção.

## Funcionalidades

- **Server-Side Rendering (SSR)**
- **Static Site Generation (SSG)**
- **API Routes**
- **File-based Routing**

## App Router
O novo modelo de roteamento baseado em arquivos dentro do diretório `app/`.

```tsx
// app/page.tsx
export default function Page() {
  return <h1>Hello, Next.js!</h1>
}
```
