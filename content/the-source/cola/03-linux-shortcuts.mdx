# Linux Shortcuts

## Navegação

- `cd`: Mudar diretório home.
- `cd ..`: Voltar um nível.
- `pwd`: Caminho atual.
- `ls`: Listar arquivos.

## Arquivos e Pastas

- `mkdir <nome>`: Criar pasta.
- `touch <nome>`: Criar arquivo.
- `rm <nome>`: Remover arquivo.
- `rm -rf <nome>`: Remover pasta recursivamente.
- `cp <origem> <destino>`: Copiar.
- `mv <origem> <destino>`: Mover/Renomear.

## Sistema

- `top`: Processos em tempo real.
- `htop`: Versão melhorada do top.
- `df -h`: Espaço em disco.
- `free -h`: Uso de RAM.
