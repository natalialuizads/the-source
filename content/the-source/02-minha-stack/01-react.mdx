# React

## Conceitos Chave

- **JSX**: Sintaxe similar a HTML no JS.
- **Components**: Blocos de construção reutilizáveis.
- **Props**: Dados passados para componentes.
- **State**: Dados internos do componente.
- **Hooks**: `useState`, `useEffect`, etc.

```jsx
import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <button onClick={() => setCount(count + 1)}>
      Clicked {count} times
    </button>
  );
}
```
